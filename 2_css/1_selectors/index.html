Les sélecteurs CSS

Par balise

Pour sélectionner toutes les instances d'un type de balise, on utilise simple le nom de la balise visée. 

        h1 {
            color: blue;
        }
        p {
            font-weight: bold;
        }
        li {
            list-style: none;
        }

Par id

Pour sélectionner un élément par son identifiant (id), on utilise le symbole # (dièse). 

        #user_1 {
            color: blue;
        }
        #titre_pincipal {
            font-weight: bold;
        }

Par classe

Pour sélectionner tous les éléments portant la même classe CSS, on utilise le symbole . (point). 

        .user {
            color: blue;
        }
        .item {
            background: red;
        }

Combiner les sélecteurs

Pour sélectionner plusieurs sélecteurs, on utilise le symbole , (virgule). 

        #titre_principal, .title, em {
            font-weight: bold;
        }
        .item, li {
            list-style: none;
        }

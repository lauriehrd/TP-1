utilisateurs windows =>
naviguez jusque c://wamp ou xampp
=> puis trouvez le dossier htdocs ou www
==> ce dossier est la racine de votre hébergement web
====> vous y placerez tout votre code

pour accéder à cet espace depuis le navigateur =>
localhost OU 127.0.0.1

ATTENTION =>

par défaut, le dossier www ou htdocs contient un index
renommez le fichier en _index pour faire apparaître les dossiers dans le navigateur, au lieu de la page d'accueil

ATTENTION BIS =>
sur windows, les extensions de fichiers sont parfois masquées par défaut, modifiez ce comportement afin de les voir

COMMANDE DU TERMINAL (LINUX OU GITBASH)

cd / (retourne à la racine du disque dur)
cd nom_du_dossier (entre dans le dossier indiqué)
cd .. (remonte d'un niveau dans le disque dur)
cd ../.. (remonte de 2 niveaux dans le disque dur)
cd ../../.. (remonte de 3 niveaux, etc....)
mkdir nom_du_dossier (make dir => créé un dossier)
ls (liste les dossiers et fichier du répertoire courant)
touch nom_du_fichier.extension (créér un fichier)
rm nom_fichier (supprime un fichier)
rmdir (supprime un dossier vide)
rmdir -rf (supprime un dossier et tout son contenu)
